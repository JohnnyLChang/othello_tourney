File specification

Students must create a file called strategy.py with a class called Strategy containing a method called best_strategy taking the arguments board, player, best_move, still_running in that order. best_move and still_running are multiprocessing.Value objects, and can be changed using best_move.value=34 for example. still_running is another multiprocessing.Value flag turned to 0 by the server when the code is about to be killed.

The board will be a length-100 string of chars from {'?', '.', '@', 'o'}, representing an 8x8 board of blanks '.' surrounded by a border '?'. The player strings are '@' for black and 'o' for white. The move is in integer representing the index in the board list of a blank space to be claimed by the current player.

Example code

To get you started, here is the most basic form of AI, random choice: 


import random
import time

class Strategy():
    # implement all the required methods on your own
    def best_strategy(self, board, player, best_move, running):
        time.sleep(1)
        if running.value:
            best_move.value = random.choice(self.legal_moves(board, player))

Multiple files

Any files students wish to upload in addition to strategy.py must have a file extension of .py, .txt, or .pkl and must be less than 20 megabytes in size.

Other Info

If you code fails to return a move, returns an invalid move, or errors for whatever reason, it will be treated as a forfeit. As such, make sure you code works on your computer before using this website to test against other ais. Just as a reminder, if your code is caught cheating during the tournament in any way, you will be caught and given an integrity violation, so just don't do it.
